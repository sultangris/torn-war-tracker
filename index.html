              Setup

              Paste your Torn API key (Limited access is enough). It stays in this browser via localStorage and is sent only to api.torn.com.

              Stat estimates use FFScouter when a key is set, and fall back to a coarse rank-based range otherwise. Your own total is fetched once a day via the Torn API (Limited access); paste it manually above if your key is Public-tier.

              Need a key? Create a Custom key with only the selections this app needs (user:basic,battlestats + faction:basic,wars,members).

              Manual enemy-faction override lives in the top bar — type a faction ID and press Track. Press Auto to resume war auto-detection.
